# PROMT_arch Проект

PROMT_arch - это Python-приложение для обработки и анализа данных с графическим интерфейсом пользователя.

## Структура проекта

### Основные файлы

- **pa.py** - Основной исполняемый файл приложения. Содержит код графического интерфейса и основную логику работы программы.

- **Descript.md** - Описание проекта в формате Markdown. Содержит подробную информацию о функциональности и возможностях приложения.

- **Overview.txt** - Обзор проекта. Краткое описание основных особенностей и целей разработки.

- **requirements.txt** - Файл зависимостей Python. Содержит список всех внешних пакетов, необходимых для работы проекта, которые можно установить командой `pip install -r requirements.txt`.

### Папки

- **ScreenShoot** - Папка с изображениями (скриншотами) интерфейса приложения. Содержит визуальные материалы для демонстрации работы программы:
  - 1.png, 2.png, 3.png, 4.png, 5.png - Скриншоты различных состояний и интерфейсов приложения.

### Файлы конфигурации проекта

- **pyproject.toml** - Современный файл конфигурации Python-проекта. Определяет метаданные проекта, зависимости, версию и параметры сборки. Используется для установки проекта командой `pip install .`.

- **.gitignore** - Файл с правилами для системы контроля версий Git. Определяет, какие файлы и папки не должны отслеживаться Git (например, файлы окружения, кэш, временные файлы).

- **LICENSE** - Файл лицензии проекта. Содержит информацию об авторских правах (Copyright) и условиях использования программного обеспечения. В данном случае проект защищен авторским правом (All Rights Reserved).

- **CONTRIBUTING.md** - Руководство для участников. Описывает процесс внесения изменений в проект, правила оформления запросов на слияние и рекомендации по разработке.

## Установка и запуск

1. Установите зависимости:
```
pip install -r requirements.txt
```

2. Запустите приложение:
```
python pa.py
```

Или установите проект как пакет:
```
pip install .
```

После установки проекта можно будет запускать командой `promt-arch` из любой директории.
